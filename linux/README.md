# Server Monitoring Script

Bash-скрипт для проверки состояния Linux-сервера.

## Возможности

- вывод имени сервера;
- вывод даты и времени;
- вывод IP-адресов;
- проверка использования памяти;
- проверка заполнения диска;
- проверка состояния Nginx;
- запись результатов в лог.

## Требования

- Linux
- Bash
- systemd
- установленный Nginx (для проверки сервиса)

## Установка

Клонировать репозиторий:

```bash
git clone https://github.com/Twody2D/devops-training1.git
```

## Запуск

```bash
chmod +x info.sh
./info.sh
```
